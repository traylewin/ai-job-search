Edge Runtime

Senior Engineer, Infrastructure

Remote (US) · Full-time

About the Role

We're building the infrastructure that powers the modern web. As a Senior Engineer, Infrastructure on the Edge Runtime team, you'll work on the systems that serve billions of requests per day with industry-leading performance. Think about the last time a website loaded instantly — that's the kind of experience we're building.

What You'll Do

  • Build and optimize our global edge computing infrastructure
  • Design systems for serverless function execution at massive scale
  • Improve cold start times and runtime performance
  • Work with V8, WebAssembly, and cutting-edge web platform APIs
  • Contribute to open-source projects (Next.js, Turbopack, SWC)
  • Collaborate with the framework team on DX improvements

About You

  • 6+ years of systems or infrastructure engineering experience
  • Strong proficiency in Rust, Go, or C++
  • Experience with edge computing, CDNs, or serverless platforms
  • Deep understanding of the HTTP stack, DNS, and TLS
  • Passion for web performance and developer tools

Compensation

$190,000 - $245,000 base + equity + benefits. Fully remote with optional access to offices in SF and NYC.
